# glyphstudios-exam

## Frontend
* Built with React using create-react-app
* Used Material UI for some styling

## Backend
* Built using Node.js with Express

## Running the App

### 1st step - Run the backend:
* Go to the root directory of the project
```
cd backend
npm install
npm start
```

### 2nd step - Run the frontend:
* Go to the root directory of the project
```
cd frontend
npm install
npm start
```
